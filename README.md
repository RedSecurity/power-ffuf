# power-ffuf
Very Fast and Powerful Web Fuzzer
## installation
1. Install Golang
2. Run the command below
```
bash installer.sh
```
## Usage
Just run the following command in the power-ffuf directory
```
./power-ffuf
```
OR
```
bash power-ffuf
```
